# 🎨 Пиксельный редактор

Простой и функциональный веб-редактор для создания пиксель-арта с возможностью экспорта в SVG и PNG.

## ✨ Возможности

- **Рисование** - рисуйте пиксель за пикселем с регулируемой кисточкой (1-32px)
- **Цвет** - встроенный color picker для выбора любого цвета
- **Ластик** - удаляйте пиксели без ограничений
- **Заливка** - инструмент заливки (flood fill) для быстрого заполнения областей одного цвета
- **История** - отмена (undo) и повтор (redo) всех действий
- **Размер сетки** - переменный размер пиксельной сетки (1-100 пикселей)
- **Экспорт в SVG** - сохраняйте рисунки как векторную графику
- **Экспорт в PNG** - сохраняйте как растровые изображения

## 🚀 Быстрый старт

1. Откройте `index.html` в браузере
2. Начните рисовать!
3. Скачайте результат в SVG или PNG

## 🛠️ Технические детали

- **HTML5 Canvas** для рисования
- **JavaScript ES6+** для логики
- **CSS3** для современного интерфейса
- История действий в памяти браузера
- SVG экспорт преобразует пиксели в элементы `<rect>`

## 🎮 Горячие клавиши и инструменты

| Инструмент | Описание |
|-----------|----------|
| Кисть | Рисуйте пиксели текущим цветом |
| Ластик | Удаляйте пиксели (белый цвет) |
| Заливка | Заполняйте области одного цвета |
| Отмена | Отмените последнее действие |
| Повтор | Вернитесь к отменённому действию |
| Очистить | Очистите весь холст |

## 📝 Примеры использования
